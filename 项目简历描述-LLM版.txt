【本草 AI·中医大模型诊疗系统】2024.03 – 2024.06
独立开发（AI Agent 辅助）

基于 31 万条真实中医医案，借助 Claude Code Agent 辅助开发了一款双模中医药诊疗 Web 应用。后端 Python Flask，前端原生 HTML/CSS/JS，设计双版本架构——规则引擎版（零依赖，基于中医知识库的模式匹配 + 症状分型 + 穴位推荐）与 LLM 版（集成 DeepSeek API，System Prompt 纳入八纲/脏腑/气血津液辨证框架、危险信号分级排查、方药安全提示与追问引导）。实现路由分离与 `force_llm` 参数区分双版共用同一 `/api/chat` 接口，LLM 可用时走大模型回复，不可用时自动降级至规则引擎。前端实现聊天式交互（打字动画、多轮上下文会话、自适应气泡布局），LLM 版采用紫色主题差异化视觉，规则版嵌入原有新中式 UI。配置 Render 一键部署，提供双链接直接在线访问。
